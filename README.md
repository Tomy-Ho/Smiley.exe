"Smiley.exe" developed at PULS GAME JAM 2025 in March 2025 in a team of 3 people.

Goal of this Game Jam: Develop a game within 48 hours with the main topic "Kaputt" (eng.: "broken").

This game can be found on itch.io https://angrybirdplayer.itch.io/smileyexe


Smiley.exe was developed in March 2025. The project was moved to this repo in January 2026.
